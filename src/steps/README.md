### 基础用法

引导用户按照流程完成任务的分步导航条，可根据实际应用场景设定步骤，步骤不得少于 2 步。

### 引入

```js
  import Vue from 'vue';
  import { Steps } from 'tanma-design';
  
  Vue.use(Steps);
```
